HOW TO PLAY

Object of the game: Make the number 24 from the four numbers
shown. You can add, subtract, multiply and divide. Use all four
numbers, but use each number only once
Example. 3 3 5 7 = 5x3-7x3
add use symbol "+"
subtract use symbol "-"
multiply use symbol "x"
divide use symbol "/"
